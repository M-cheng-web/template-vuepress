# vuepress模板

## 说明
此模板是快速用 `VuePress + Github Pages` 搭建博客/文档系统

## 启动
```
yarn & yarn dev
```

## 部署
会在你的 github 项目仓库中创建一个分支或者在目标分支中提交
``` 
windows: 在项目根路径进入`Git Bash`后执行 - `sh deploy.sh`
macOs: 直接执行 - `sh deploy.sh`
```

需要在 GitHub 目标仓库中 settings > pages 中设置
